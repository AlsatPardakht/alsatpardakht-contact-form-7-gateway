== Description ==

**Contact Form 7 ** این افزونه شما را قادر می سازد تا براحتی اقدام به ایجاد درگاه پرداخت اینترنتی برای فرم های خود کنید

== Installation ==

* قبل از همه در وبسایت آل‌سات پرداخت ثبت نام کرده و پس از درخواست درگاه و ارسال مدارک افزونه را دانلود، نصب و فعال سازی افزونه از قسمت پیکربندی ووکامرس -> پرداخت ها -> پرداخت امن آل‌سات پرداخت -> مدیریت تکمیل کنید، سپس میتوانید.
* تنظیمات افزونه را به شکل دلخواه خود تغییر دهید.
* 1 : در وبسایت آل‌سات پرداخت ثبت نام کنین و مراحل احراز هویت رو تکمیل کنین.
* 2 : افزونه رو بارگزاری و فعال کنین.
* 3 : از قسمت فرم تماس -> تنظیمات آل‌سات پرداخت ، تنظیمات مورد نظر خودتون رو اعمال کنین.
* 4 : به فرم تماس خودتون رفته و در قسمت اطلاعات پرداخت قیمت مورد نظر خودتون رو اعلام کنین.
* 5 : در آمد بیشتری کسب کنین و لذت ببرین.
